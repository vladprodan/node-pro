# Структура проекту

## Огляд структури

```
src/
├── config/                      # Конфігурація додатку
│   ├── app.config.ts           # Налаштування додатку (порт, env, URL)
│   └── configuration.ts        # Загальна конфігурація
│
├── users/                       # Модуль користувачів
│   ├── users.controller.ts     # API endpoints для користувачів
│   ├── users.module.ts         # Модуль з усіма залежностями
│   └── users.service.ts        # Бізнес-логіка роботи з користувачами
│
├── app.controller.spec.ts      # Тести для головного контролера
├── app.controller.ts           # Головний контролер додатку
├── app.module.ts               # Кореневий модуль (збирає все разом)
├── app.service.ts              # Головний сервіс додатку
└── main.ts                     # Точка входу в додаток
```

## Опис папок

### `/config`

Тут зберігаються всі налаштування додатку.

### `/users`

Модуль, який містить все необхідне для роботи з користувачами:

- **Controller** - приймає запити з фронту
- **Service** - вся логіка (створення, оновлення, видалення юзерів)
- **Module** - збирає контролер і сервіс разом
